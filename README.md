# Cross_Validation
Aqui você irá encontrar a atividade do módulo 35 do curso de Cientista de Dados da Ebac.

A atividade consiste em aplicar técnicas de validação cruzada utilizando o modelo Random Forest para a classificação de focos de incêndio, avaliando a capacidade de generalização do modelo.

Durante a atividade serão abordados temas como:

* Validação cruzada
* Modelo Random Forest
* Classificação supervisionada
* Avaliação de desempenho de modelos

O objetivo é compreender a importância da validação cruzada na avaliação de modelos de Machine Learning, garantindo resultados mais confiáveis e reduzindo o risco de overfitting durante o treinamento.
